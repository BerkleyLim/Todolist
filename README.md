# Todolist
리액트 기반 투두리스트 만들기

# 목표
- 리액트 실력을 올리기 위해 시작
- 반응형 웹을 학습을 하기위해 제작
- 차후 스타벅스 클론코딩 진행을 위해 연습을 진행
- 리액트 뿐만 아닌 css도 연습을 위해 진행

# 사용기술
