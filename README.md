# Todolist
리액트 기반 투두리스트 토이 프로젝트 만들기

# 목표
- 리액트 실력을 올리기 위해 시작
- 반응형 웹을 학습을 하기위해 제작
- 차후 스타벅스 클론코딩 뿐만 아닌 지금까지 하고 있는 프로젝트에 개선을 진행을 위해 연습용 토이 프로젝트
- 리액트 뿐만 아닌 css도 연습을 위해 진행

# 진행기간
2023년 6월 5일 - 2023년 6월 14일

# 사용기술
React, Redux, reactstrap, scss
